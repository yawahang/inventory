# Inventory Management
Inventory Management Project

## Technologies Used
  - Database: SQL Server 2017 / SSMS 2018
  - Backend: Dot Net Core 3.1.301
  - ORM: Dapper
  - Frontend: Angular
  - UI Framework: KendoUI (Telerik), Angular Material
  
  ## Features
    - Login
    - Add/Edit Product
    - Product listing
    - Product Filtering
    - Order Report (Basic Grid/Spreadsheet Report)
    - Dashboard (Basic)
